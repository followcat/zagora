# Bundled Fonts

This Android module bundles the following fonts for terminal rendering quality:

- `jetbrainsmono_regular.ttf`
  - Source: JetBrains Mono
  - URL: https://github.com/JetBrains/JetBrainsMono
  - License: SIL Open Font License 1.1

- `jetbrainsmono_nerd_regular.ttf`
  - Source: Nerd Fonts (JetBrains Mono No Ligatures, Nerd Font Mono Regular)
  - URL: https://github.com/ryanoasis/nerd-fonts
  - License: SIL Open Font License 1.1

These files are included to improve terminal glyph coverage on Android devices.
